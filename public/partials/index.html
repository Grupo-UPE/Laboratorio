Publicar un nuevo texto


Explicacion

Tenemos nodejs y mongodb en el servidor. Desde alli solo manejamos peticiones REST. O algo similiar a ellas.

Definimos las siguientes rutas:

  • app.get('/REST/texto', texto.index)
  • app.get('/REST/texto/:id', texto.show_edit)
  • app.post('/REST/texto', texto.update)
  • app.post('/REST/delete-texto', texto.remove)
  • app.post('/REST/create-texto', texto.create)
    • Estas urls las va a manejar nodejs. Todas las demas las manejara angularjs.

    Entre los directorios de nodejs tenemos uno que se llama public. Alli es donde metimos todo lo que necesitamos para que funcione angularjs. De esta forma todas las urls que no coincidan con las definidas en app.js intentaran resolverse desde el directorio public

    Angular ahora esta manejando solo urls del index del tipo /editar/id. Ademas provee funcionalidad para crear, eliminar, listar y actualizar diferentes textos.

    La idea seria contar con algun sistema de autenticacion y verificacion de permisos para que un usuario solo pueda editar/borrar sus propios textos.

    Como prueba de concepto de lo que se puede hacer con node+mongo+express+angular me parece que esta bien. Estamos mas cerca de crear una aplicacion MEAN con todas las letras.
